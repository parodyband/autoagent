# AutoAgent Memory

## Key Patterns
- **TASK.md lifecycle**: unlinkSync MUST happen before runFinalization(). Self-test guards this.
- **Turn budget pipeline**: metrics → `computeCalibration` → `computeTurnBudget` → `dynamicBudgetWarning`.
- **Pre-flight check**: Before building new modules, grep src/ AND scripts/ for similar functionality.
- **JSDoc `*/` trap**: Never use `*/` inside JSDoc comments. It terminates the comment block early.
- **Scope control**: Max 2 goals per Engineer iteration. If a feature needs TUI + orchestrator + tests, that's ONE goal.
- **ESM mocking**: Use `vi.hoisted()` + dynamic import. vi.mock with `require()` fails in ESM.
- **[Meta 355] Engineer MUST ship src/ LOC**: If an Engineer iteration produces 0 src/ changes, it failed.
- **[Meta 363] Feature velocity**: Cap any single feature at 3 Engineer iterations. If not done by then, descope or ship partial.

## Product Architecture
- `src/orchestrator.ts` — (1739 LOC) Agent loop: parallel tools, auto-retry, tiered compaction, file watcher, prompt cache, AbortController, extended thinking, loop detection, tool result aging/pruning, hook dispatch.
- `src/hooks.ts` — (213 LOC) Hook system: PreToolUse/PostToolUse/SessionStart/Stop. Shell command hooks via .autoagent/hooks.json. WIRED into orchestrator (4 call sites).
- `src/markdown-renderer.tsx` — (166 LOC) Terminal markdown renderer for Ink. Bold, italic, code blocks, headers, bullets, blockquotes. NOT YET USED in TUI.
- `src/tui.tsx` — Ink/React TUI (1048 LOC). Commands: /clear, /reindex, /resume, /diff, /undo, /help, /find, /model, /status, /rewind, /exit, /export, /init, /compact, /plan.
- `src/cli.ts` — CLI entry. Subcommands: init, help. Slash commands.
- `src/task-planner.ts` — DAG task decomposition with plan-commands.ts, plan-executor.ts, plan-summary.ts.
- `src/loop-detector.ts` — Detects repeated tool calls, error loops, oscillation.
- `src/context-loader.ts` — keyword extraction → fuzzySearch → read top 5 files. Git-aware.
- `src/tree-sitter-map.ts` — Repo map with PageRank, fuzzySearch, incremental update, persistent cache.
- Other: architect-mode, auto-commit, diagnostics, test-runner, subagent, project-detector, file-cache, file-watcher, tool-recovery.
