## Product Architecture
- `src/tui.tsx` — Ink/React TUI. Streaming, tool calls, model badge, footer. Commands: /clear, /reindex, /resume, /exit.
- `src/orchestrator.ts` — Pipeline: route model → decompose tasks → architect plan → agent loop → verify. Cost tracking. Context compaction at 150K tokens.
- `src/architect-mode.ts` — `needsArchitectMode()` detects complex requests, `generateEditPlan()` produces structured plan, injected as prefilled assistant message.
- `src/session-store.ts` — JSONL under `~/.autoagent/sessions/{project-hash}/`. Auto-clean 30 days.
- `src/project-memory.ts` — CLAUDE.md hierarchy + write-back via `saveToProjectMemory`.
- Model routing: keyword-based (CODE_CHANGE → sonnet, READ_ONLY → haiku).
