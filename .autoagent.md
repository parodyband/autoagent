1. Tool result clearing is "lowest-hanging fruit" compaction — we already do this (micro-compact + proactive summarization).
2. Context is finite with diminishing returns (n² attention). Smaller high-signal context > large noisy context.
3. Hybrid retrieval: upfront loading (CLAUDE.md) + just-in-time tools (grep/glob). We do this.
4. Structured note-taking for long-horizon tasks — we have memory.md + scratchpad.
5. Sub-agent architectures for parallel exploration — we have subagent tool.
Gap identified: We don't surface context efficiency metrics to users (tokens/turn, context utilization ratio).
